# Changelog

## v0.2.1
- 修复：文件大小格式化对 None/非法值的兼容，避免运行时报错。
- 修复：空搜索结果列表渲染时区间显示异常（1-0）。
- 修复：补齐 /settings 命令入口，并在启动时注册机器人命令菜单。
- 优化：services 模块不再在导入时强制配置日志；Redis getdel 降为 debug 日志。
- 测试：Redis 回退测试使用独立事件循环，减少不同 Python 版本兼容问题。
- 文档：修复 README 标题乱码；新增 requirements-dev.txt 指引。

## v0.2.0
- 搜索结果页：新增页码快捷与页码选择器；新增筛选/排序区块。
- 筛选：分级/格式/体积/字数筛选（基于 Meilisearch filter）。
- 排序：最热/最新/最大排序（基于 downloads/created_at/file_size）。
- 设置：二级菜单（内容分级/搜索按钮模式）+ 多项开关（匿名上传、静默反馈等）。
- 文案：列表头部与条目二行展示优化，对齐与转义增强。
